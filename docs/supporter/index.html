Supporter Subscription

ありがたいことに、これまで「Jasper便利だからお金払いたい」「Trickleの開発応援したいから投げ銭したい」などの声を何度かいただきました。

そこで、サポーターサブスクリプションというものを販売することにしました。
サポーター サブスクリプションは収益化が目的ではなく、開発しているプロダクトの運営費（サーバ代・ドメイン代など）をサポートしていただくことが目的です。 これらの費用を少しでもサポートしてもらえると大変嬉しいです🙏

また、「プロダクトを応援してあげよう！」という方も是非よろしくお願いします。大変励みになります。

決済はrebuild.fmとおなじくGumroadを利用しています。クレジットカードまたは PayPal を利用してお支払いいただけます。サブスクリプションはGumroadから、いつでもキャンセルすることができます。


なお、投げ銭ではなく商取引とするために、「このページに名前を掲載する権利を販売する」という形にしています。 なので、購入後に「ライセンス番号」「サイトへの表示名」「リンク先URL」「アイコン画像もしくはURL」をh13i32maru+supporter@gmail.com までご連絡ください。 掲載を望まない方は連絡不要です。

※ 投げ銭やドネーションにしなかったのはこちらの記事を参考にしました。

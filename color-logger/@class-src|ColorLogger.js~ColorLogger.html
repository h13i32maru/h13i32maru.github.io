ColorLogger

display colorful log. now, not support browser.

format:
[LogLevel] [Time] [File] log text

format with tag:
[LogLevel] [Time] [File] [Tag] log text

log level and color:

  • verbose: purple
  • debug: blue
  • info: green
  • warning: yellow
  • error: red

Example:

import Logger from 'color-logger'

// simple usage
Logger.v('verbose log');

// tag usage
let logger = new Logger('MyTag');
logger.d('debug log');
